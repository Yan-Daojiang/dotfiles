# Getting Started
```
git clone https://github.com/Yan-Daojiang/dotfiles.git
```

<details>
<summary>try it in docker:</summary>

```bash
# current directory is dotfiles
docker build -t my-dotfiles -f ./docker/Dockerfile .
docker run --rm -it my-dotfiles
```

</details>

## Install
``` 
cd dotfiles
./install
```
